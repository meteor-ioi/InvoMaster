# UI 布局修改方案预览

以下方案基于当前的“左-中-右”三栏布局进行功能增强建议。

---

## 方案 1：工具栏平铺（迭代式增强）
**特点**: 在中间预览区的顶部工具栏直接平铺核心参数，右侧列表增加过滤。适合快速切换。

```text
+---------------------------------------------------------------------------------------+
| Header: 影刀 离线单据识别 (Theme Switcer)                                              |
+------------+-----------------------------------------------------------------+--------+
| [左: 模板] | [中: 预览板块]                                                  | [右: ] |
|            | +-------------------------------------------------------------+ | [要素] |
| 模板列表   | | [图标] 策略: [⚖️ 平衡 v]  置信度: [---O---] 0.25  [重新识别]   | | [编辑] |
| (过滤/搜索)| | 缩放: [ 100% ] [眼睛] 显示预览                              | |        |
|            | +-------------------------------------------------------------+ | 字段名 |
| [ ] 模板 A | |                                                             | | [文本] |
| [ ] 模板 B | |                                                             | |        |
|            | |                      PDF 预览画布                           | | +----+ |
|            | |                                                             | | |过滤| |
|            | |                                                             | | |[x]T| |
|            | |                                                             | | |[ ]H| |
|            | +-------------------------------------------------------------+ | +----+ |
+------------+-----------------------------------------------------------------+--------+
```
- **中间项**: 顶部增加“策略”下拉框。
- **右下角**: 悬浮或内嵌一个“类别过滤器”小磁贴，勾选只看/不看特定类型。

---

## 方案 2：分层控制台（操作解耦）
**特点**: 将“识别设置”与“预览工具”分离，在预览区上方建立一个专门的逻辑控制层。

```text
+---------------------------------------------------------------------------------------+
| Header (带关键状态显示)                                                               |
+------------+-----------------------------------------------------------------+--------+
| [左: 模板] | [中: 预览板块]                                                  | [右: ] |
|            | +--- 识别配置 (Identification Settings) ----------------------+ | [要素] |
|            | | 模式: ( )极速  (●)平衡  ( )精细  | 分辨率: [1024 v]  IoU: 0.45 | | [编辑] |
|            | +-------------------------------------------------------------+ |        |
|            | +--- 视图工具 ------------------------------------------------+ |        |
|            | | [ 缩放 ] [ 撤销 ] [ 重做 ] [ ]只看表格 [ ]不看页眉页脚      | |        |
|            | +-------------------------------------------------------------+ |        |
|            | |                                                             | |        |
|            | |                      PDF 预览画布                           | |        |
|            | |                                                             | |        |
+------------+-----------------------------------------------------------------+--------+
```
- **逻辑层**: 显式展示“模式、分辨率、IoU”，点击不同模式时下方滑块/值自动联动。
- **视图层**: 将类别过滤开关与缩放按钮放在一起，强调“这只是影响视觉展示”。

---

## 方案 3：高级侧边栏 Tab（垂直整合）
**特点**: 将复杂的参数隐藏在侧边栏的 Tab 切换中，给中间留出最大的视觉空间。

```text
+---------------------------------------------------------------------------------------+
| Header                                                                                |
+------------+-----------------------------------------------------------------+--------+
| [左: 模板] | [中: 预览板块]                                                  | [右: ] |
|            | +-------------------------------------------------------------+ | [编辑] |
| [列表][⚙️]  | | [ ] 缩放 100%  [ ] 重新识别  [ ] 状态: 就绪/匹配成功         | | [参数] |
|    ^--切换 | +-------------------------------------------------------------+ |   ^--  |
|            | |                                                             | | [识别] |
| [设置面板] | |                                                             | | [设置] |
| 1. 策略选择| |                      PDF 预览画布                           | | [IOU ] |
| 2. 详细参数| |                                                             | | [img ] |
|    IoU: ---| |                                                             | | [ ]表| |
|    Img: ---| |                                                             | | [ ]脚| |
+------------+-----------------------------------------------------------------+--------+
```
- **左侧 Tab**: 将原有的“模板列表”变更为双标签，第二个标签为“识别参数配置”。
- **右侧 Tab**: 或在右侧“要素编辑”旁增加“渲染/过滤”选项盒。
- **中间**: 极致简洁，仅保留最核心的动作按钮。

---

## 汇总建议
- **短期改进**: 推荐 **方案 1**。改动最小，直接在顶部已有的工具槽位插入新组件。
- **长期体验**: 推荐 **方案 2**。工业单据的“识别配置”与“字段校对”是两个独立心智过程，物理分割有助于用户专注。
- **过滤功能**: 无论采取哪种方案，建议将 **“类别过滤”** 放在画布右侧或下方，因为它与校对关联最深。
